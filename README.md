# Munyeshya Beni Obed Portfolio

Personal portfolio built with React, Vite, Bootstrap, Framer Motion, and React Router.

## Stack

- React
- Vite
- Bootstrap 5
- Framer Motion
- React Router

## Local development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```

## Deploying to Vercel

This project is configured as a static Vite app and includes a rewrite so direct visits to routed pages such as `/projects/factory-management-system` still load correctly.

Recommended Vercel settings:

- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`
